# nian1123.github.io

基于 `Hexo` 与 `hexo-theme-stellar@1.33.1` 的个人博客站点。

## 本地开发

```bash
npm install
npm run server
```

## 构建

```bash
npm run build
```

## 部署到 GitHub Pages

1. 在 GitHub 创建仓库：`nian1123/nian1123.github.io`
2. 将本目录推送到该仓库的 `main` 分支
3. 在仓库 `Settings -> Pages` 中确认使用 `GitHub Actions`
4. 后续每次推送到 `main` 都会自动部署
